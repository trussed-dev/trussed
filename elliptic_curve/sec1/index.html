SEC1 encoding support.

Support for the Elliptic-Curve-Point-to-Octet-String encoding described in SEC1: Elliptic Curve Cryptography (Version 2.0) section 2.3.3 (p.10):

https://www.secg.org/sec1-v2.pdf
