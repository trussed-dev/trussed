Struct elliptic_curve::ScalarBytes

pub struct ScalarBytes<C: Curve> { /* private fields */ }
Expand description

Scalar bytes: wrapper for FieldBytes which guarantees that the the inner byte value is within range of the Curve::ORDER.

Does not require an arithmetic implementation.
