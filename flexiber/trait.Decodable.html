Expand description

Decoding trait.

Decode out of decoder, which essentially is a slice of bytes.

One way to implement this trait is to implement TryFrom<TaggedSlice<'_>, Error = Error>.

It should evaluate to the desired tag via Tag::try_from(byte)?.
