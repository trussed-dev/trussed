Struct flexiber::Tag

pub struct Tag {
    pub class: Class,
    pub constructed: bool,
    pub number: u16,
}
Expand description

The tag field consists of a single byte encoding a tag number from 1 to 254. The values ‘00’ and ‘FF’ are invalid.
